# Crud-Contatos

Esse projeto foi criado com C#, .NET, Javascript, Jquery, HTML/CSS, Bootstrap e SQL Server.

A aplicação consiste em um Sistema de Cadastro de Contatos com a possibilidade de cadastrar, editar e apagar contatos.
